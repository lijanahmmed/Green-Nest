# Green Nest

**Green Nest** is a modern and eco-friendly web application dedicated to plant lovers. It’s designed to help users explore, learn, and buy indoor and outdoor plants while promoting sustainable living. Built with React and Firebase, this project focuses on a clean UI, smooth navigation, and user-friendly experience.

---

## Features

- **Plant Catalog** – Browse a variety of indoor and outdoor plants with detailed information.
- **Plant Care Tips** – Get expert advice on watering, sunlight, and fertilization.
- **Meet Our Green Experts** – Learn from plant specialists and get personalized consultation.
- **Plant Booking System** – Users can book plants with instant feedback and confirmation.
- **Authentication System** – Secure login, register, and password reset using Firebase Auth.
- **Contact & Feedback** – Users can reach out through a simple contact form.
- **Responsive Design** – Works perfectly on mobile, tablet, and desktop devices.

---

## Tech Stack

- **React.js** – Frontend framework
- **Firebase** – User authentication & hosting
- **Tailwind CSS** – Modern styling and responsive design
- **React Router** – Page navigation
- **Swiper.js / Framer Motion** – Smooth animations and sliders
- **React Toastify** – Toast messages and alerts

## Clone the Repository

Use the following command to clone the project:

```bash
git clone https://github.com/lijanahmmed/Green-Nest.git

cd Green-Nest
```

### Install Dependencies

```bash
cd Green-Nest
npm install

```

### Setup Firebase

```
VITE_APIKEY=your_firebase_api_key
VITE_AUTHDOMAIN=your_firebase_auth_domain
VITE_PROJECTID=your_firebase_project_id
VITE_STORAGEBUCKET=your_firebase_storage_bucket
VITE_MESSAGINGSENDERID=your_sender_id
VITE_APPID=your_app_id
```

### Start development server

```bash
npm run dev
```
